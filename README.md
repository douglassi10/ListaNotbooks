
# Web Scraping ListaNotbooks

Robô Web Scraping navega até uma página web, filtra dados de uma lista de notbooks de marca especifica e organiza esses dados em um arquivo .json 

## Tecnologias Utilizadas

- Python
- Json
- Selenium WebDriver
- IDE VScode
